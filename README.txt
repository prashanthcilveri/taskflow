TaskFlow - Team Task Manager

Live URL:
https://web-production-4a25.up.railway.app

GitHub Repository:
https://github.com/prashanthcilveri/taskflow

Project Overview:
TaskFlow is a full-stack task management web application where users can create projects, assign tasks, track progress, and manage workflows using role-based access control.

Features:
- User Signup/Login Authentication
- Role-Based Access (Admin/Member)
- Project Management
- Task Creation and Assignment
- Task Status Tracking
- Dashboard for managing tasks
- Responsive UI

Tech Stack:
- Python
- Flask
- SQLite
- HTML/CSS
- Bootstrap
- Railway Deployment
- GitHub

Deployment:
The application is deployed live using Railway.

How to Run Locally:
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
